Detailed Description

Class with all functions for Fast Fourier Transform. Consists of the main functions for FFT, iFFT and FourierFilter. Has a next private fields:

Parameters
mFourierSignalclass vector<comp> - input signal
mFourierSignalCleanclass vector<comp> - input signal after filtering
mFourierFrequencyclass vector<comp> -input frequencies (obtain after transforming)
mFourierFrequencyCleanclass vector<comp> - frequencies after filtering
